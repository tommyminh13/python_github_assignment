# Gym Time Calculator

This simple Python program helps users estimate how many hours they will spend at the gym each week. The user enters how many hours they plan to go to the gym today, and the program multiplies it by seven to calculate weekly gym hours. This tool is helpful for anyone trying to plan or track their fitness routine.

---

How to Run

Make sure you are in the project folder, then run:

python app.py

or

python3 app.py

---
Example Interaction

Welcome to the Gym Time Calculator!
How many hours do you plan to go to the gym today? 2
If you keep this pace, you will spend 14.0 hours at the gym this week!
